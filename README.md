# Christmas Gift Reveal Game

A beautiful, interactive Christmas gift reveal game built with HTML5 Canvas, Tailwind CSS, and vanilla JavaScript.

## Features

- **Scratch-to-Reveal Game**: Interactive canvas-based scratch card where users can "unwrap" their gift
- **Christmas Theme**: Romantic Christmas design with snowflakes, festive colors, and sweet animations
- **Chocolate Rain Animation**: Falling chocolate emojis during the reveal sequence
- **Multi-Stage Reveal**: Progressive reveal with heartfelt messages and final gift details
- **Mobile Responsive**: Optimized for both desktop and mobile devices
- **Touch & Mouse Support**: Works with both touch gestures and mouse interactions

## How It Works

1. **Welcome Screen**: Displays profile photo with rotating Christmas border and heartbeat animation
2. **Scratch Game**: User scratches a festive wrapping paper overlay to reveal the gift underneath
3. **Progress Tracking**: Shows percentage of wrapping paper removed
4. **Auto-Reveal**: Automatically completes reveal when 60%+ is scratched
5. **Prize Sequence**:
   - Stage 1: Romantic opening message
   - Stage 2: Sweet teaser
   - Stage 3: Chocolate rain animation
   - Stage 4: Final reveal with bonbon workshop details and sparkle effects

## Technologies

- HTML5 Canvas for scratch effect
- Tailwind CSS for styling
- Vanilla JavaScript for all interactions
- CSS animations for visual effects
- Google Fonts (Dancing Script, Poppins)

## Setup

Simply open `index.html` in a web browser. No build process or dependencies required!

## Customization

To customize for your own gift reveal:

1. Replace `patti_new.jpg` with your recipient's photo
2. Edit the messages in the HTML (search for `stage-1`, `stage-2`, `stage-3`)
3. Modify colors in the CSS (search for `#B91C1C`, `#047857`, `#FFC107`)
4. Change the gift details in the final reveal section
